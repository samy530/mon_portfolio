Détection des Émotions en Temps Réel

Application Python utilisant la webcam pour détecter les émotions faciales en temps réel avec un modèle Deep Learning. Les visages sont encadrés avec des couleurs différentes selon l’émotion détectée.

  • ✅ Détection faciale avec Haar Cascade
  • ✅ Prédiction des émotions avec un modèle Keras pré-entraîné
  • ✅ Support multi-personnes
  • ✅ Affichage des émotions avec rectangles colorés

UniForTech - Plateforme pédagogique

Plateforme web collaborative pour le département informatique de l’Université Mouloud Mammeri Tizi-Ouzou, facilitant l’accès et le partage de contenus pédagogiques (cours, TD, TP, mémoires, livres, annonces…).

  • 👀 Pour le Visiteur : navigation libre, inscription via formulaire.
  • 🎓 Pour l’Étudiant : authentification sécurisée, consultation et publication de ressources, gestion de profil, contact admin, déconnexion sécurisée.
  • 👨‍💼 Pour l’Administrateur : gestion des comptes, publications, bibliothèque, messages et statistiques.
